Interactive Trachea Model with Detail Images

How to open:
1. Extract this ZIP file.
2. Open the folder.
3. Double-click index.html.
4. Use Google Chrome or Microsoft Edge.
5. Click the orange dots.

How to add different images for each part:
1. Put your images in the same folder as index.html.
2. Rename them exactly as:
   - detail_larynx.png
   - detail_thyroid.png
   - detail_trachea.png
   - detail_cartilage.png
   - detail_carina.png
   - detail_right_bronchus.png
   - detail_left_bronchus.png
   - detail_bronchioles.png
3. Refresh the browser.

Note:
The current template includes your uploaded detail image.
You can replace each detail image file with the correct diagram for that anatomy part.
